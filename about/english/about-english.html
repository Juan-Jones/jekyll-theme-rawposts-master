    Hello,

    Rawposts is a free Jekyll portfolio-style theme, which is designed to be a starting point for any Jekyll website. By using this theme, the users can deploy a personal blog or website on Github Pages quickly.

    Features:

    • Fully responsive
    • Optimized for tablets & mobiles
    • Link to social media (Github/Facebook/Twitter/Linkin)
    • Article sharing to social media (Linkin/Fackbook/Twitter/Wechat...,180+)
    • Sidebar for auto TOC generator
    • Excerpt
    • Code highlighter (Rouge - 20+ themes)
    • Search keyword of website
    • Article reading statistics (Google Analytics/Busuanzi)
    • Statistics of site visits (Google Analytics/Busuanzi)
    • Pagination
    • Archive posts
    • Category posts
    • Comments (Gitalk)
    • RSS/Sitemap
    • SEO optimization
    • Auto build/deploy with Travis CI
    • Statistic for words count
    • jemoji for emoji
    • Latex formula support
    • dark mode selectable
    This theme is customizable (fonts, colors, layout elements, code coloration, etc...) and all the above features are defined in _config.yml file. The users can turn on/off these features only by setting the keywords in _config.yml easily.

    For the installation & usage guidance, please refer to [RawPosts - A theme for Jekyll]
